1. Use runServer-PgSqlLocalAll.sh - To run an application server that will only allow local client connections.

2. Use runServer-PgSqlRemoteClients.sh - To run an application server that will allow remote clients to connect to the database. 
	- Edit the bash file to provide the path to your database connection file and the connection information id to use.
	
	See <http://www.eclipse.org/osee/documentation/configuration/dbconnection.php> for more information.
