Index Search

What it is

An option to search for artifacts with particular tags on the default branch. The * wildcard is not supported.

How to do it

Type in all of the desired tags. Make sure the Index option is selected under Search Type, then press the Search button.

The index searching can be tightened or loosened with the Case-sensitive and Partial Match check boxes.

Shortcuts

Pressing enter in the text entry box is equivalent to pressing the Search button.
