Create new VUE workflow diagram

Purpose

To create a new VUE diagram that ATS uses to move an Action through it's specific workflow.

How to do it

  • Diagram must have Endorse and Completed
  • Endorse state must have Title, Description, Priority, Deadline, Change Type, Validation
  • Each state must have a unique identifier as the state id
  • Use "-DAtsUseWorkflowFiles" flag to instruct ATS to use a specified file in the workspace to use (versus a skynet artifact). This helps speed the development of the workflow.
  • If using "-DAtsUseWorkflowFiles" flag, put file path resolution info in AtsWorkFlowFactory.java
